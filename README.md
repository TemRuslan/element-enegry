## Важно: ссылки и `<base>`

Страницы лежат в подпапках, но при этом их можно открывать как через локальный сервер, так и напрямую с диска.
Для этого на каждой странице в `<head>` вставляется небольшой скрипт, который рассчитывает путь до корня сайта и
добавляет `<base href="...">`.

Практическое правило: внутренние ссылки в разметке пишем как пути от корня репозитория (например, `projects/index.html`,
`industries/promyshlennost/index.html`), чтобы они корректно работали с любого уровня вложенности.

## Файлы

### index.html
Основная разметка сайта с:
- Hero секцией
- Секцией отраслей применения ГПУ
- Каталогом продукции
- Слайдером услуг
- Формой обратной связи

### components/header.html
Навигационное меню с:
- Логотипом компании
- Выпадающими меню для разделов
- Мобильным меню

### components/footer.html
Нижняя часть сайта с:
- Копирайтом
- Ссылками на политику конфиденциальности
- Социальными сетями

### css/styles.css
Все стили сайта включая:
- CSS переменные для цветов и эффектов
- Стили навигации и выпадающих меню
- Glassmorphism эффекты для карточек
- Адаптивный дизайн для разных экранов
- Стили слайдера услуг

### js/script.js
JavaScript функционал:
- HTML компоненты header и footer встроены в код для работы без сервера
- Автоматическая вставка header и footer при загрузке страницы
- Функция `scrollServices()` для управления слайдером услуг

## Запуск проекта

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере.

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с CSS Grid и Flexbox
- **Tailwind CSS** - утилитарный CSS фреймворк
- **JavaScript ES6** - интерактивность
- **Font Awesome** - иконки
- **Google Fonts** - шрифты Inter и Poppins
